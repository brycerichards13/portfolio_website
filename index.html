Fantanofy

A fullstack web application that I built from scratch, which uses Spotify's API to compare their top tracks and albums to albums reviewed by popular critic Anthony Fantano.

The user logs in in with their Spotify, then my server makes a request to the Spotify API to get the data, compares the user's top tracks and albums to Anthony Fantano's reviews using a MySQL database created from thousands of webscraped reviews, and gives a score based on how similar the user's music taste is to Anthony Fantano's.

It also allows the user to view all the scores that Anthony Fantano has given the user's top albums and tracks.

Lastly, since I had an entire database of all of his reviews, I built a stats page which visualizes his different statistics in an engaging and interesting way, which I processed using the Pandas data analysis Python library.

I created the backend written in Node.js with the Express framework, and the frontend written in responsive HTML, EJS, CSS, and JavaScript. Lastly I deployed the website to AWS on a Linux server with Nginx, added SEO and SSL certification, and it has been used by hundreds of people around the world.

Celebrity Text To Speech

This python project integrates neural network models I trained utilizing the PyTorch TacoTron 2/Wavenet architecture for voice synthesis from text.

It includes an interactive GUI, which allows users to input their own text and have it be read out by a chosen celebrity.

Furthermore, it integrates functionality of the Streamlabs API, a popular software platform that live streamers use for managing their stream. My program allows the user to connect to the Streamlab's API to have the live streamers donations be read out by a chosen celebrity.

In order to create the training data for the celebrities' voices, I gathered large amounts of clear audio from online sources and split them into hundreds of distinct clips. Then, I transcribed these clips into a CSV file and trained the neural network on a Linux machine using an NVIDIA GPU. I then used these models for generating the speech from the text.

The GUI is created using a Python library called Tkinter. The program separates the multithreaded GUI application and the voice generation into two separate processes using multiprocessing.
